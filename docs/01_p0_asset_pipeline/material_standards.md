# 素材规范

本规范用于“角色资产化准备阶段”。当前目标是整理素材、建立筛选标准和形成可复用数据结构；不训练模型、不下载模型、不上传真实图片或视频。

## 角色身份 LoRA 素材标准

角色身份 LoRA 的素材只回答一个问题：这是不是林薇。它不应该承担服装、场景、姿势和画风的全部控制。

适合进入身份 LoRA 候选集的图片：

- 脸部清晰，五官、脸型、眼镜和发际线可辨认。
- 角色年龄感、气质、脸型稳定。
- 黑框眼镜完整可见，镜片反光不遮挡眼睛。
- 长深棕色头发特征清楚，发型与目标设定一致。
- 包含正脸、三分之二侧脸、轻微侧脸。
- 包含近景头像、上半身、半身，少量高质量全身图。
- 背景不抢主体，没有明显水印、字幕、贴纸或 UI 遮挡。
- 同一视频连续帧不要大量重复，只保留代表性帧。

最低建议：

- 快速验证：20 到 30 张精选图。
- 稳定实验：40 到 80 张精选图。
- 后续生产：80 到 150 张精选图。

## FaceID 素材标准

FaceID 素材用于临时身份迁移或模型路线测试，要求比普通参考图更偏向脸部质量。

最低要求：

- 1 到 3 张清晰正脸或轻微侧脸。
- 人脸占画面比例足够大。
- 脸部无遮挡，眼镜边框完整。
- 光线均匀，无严重过曝、欠曝或彩色灯污染。
- 表情自然，避免夸张张嘴、闭眼、挤眼。

推荐要求：

- 5 到 10 张高质量脸部图。
- 包含正脸、三分之二侧脸、轻微侧脸。
- 发型、眼镜、妆容尽量接近目标角色设定。
- 不混入年龄感、发型或妆容差异过大的图片。

## 服装参考素材标准

服装参考图用于生成时控制衣服结构、颜色、层次和配饰，不一定适合训练身份 LoRA。

适合放入 `datasets/linwei/outfit_refs` 的图片：

- 上衣、裙裤、外套、领口、袖口、材质和颜色清晰。
- 可以没有清晰脸，但服装主体必须完整。
- 最好包含正面、侧面、背面或局部细节。
- 服装图不要和身份 LoRA 混用，避免模型把固定衣服误学成角色身份。

鞋包、首饰、眼镜备用款、围巾等细节放入 `shoes_accessories`。

## 发型/妆容素材标准

发型素材用于控制发长、刘海、卷直、扎发、发量轮廓和发色。妆容素材用于控制眼妆、唇色、腮红、肤色质感和正式程度。

发型参考适合放入 `hairstyle_refs`：

- 发型轮廓清楚。
- 发长、刘海、分缝、扎发方式可辨认。
- 头发没有被帽子、大面积阴影或背景严重干扰。

妆容参考适合放入 `makeup_refs`：

- 眼妆、唇色、腮红或肤色质感清楚。
- 光线真实，不被滤镜或美颜严重改变。
- 妆容是可复用设定，而不是一次性特殊效果。

## 视频抽帧筛选标准

视频抽帧只产生候选素材，不能直接当训练集。

推荐流程：

1. 原视频放入 `datasets/linwei/raw_videos`。
2. 使用 `scripts/extract_keyframes.py` 默认每秒 1 帧抽到 `datasets/linwei/extracted_frames`。
3. 删除模糊、压缩噪点、闭眼、遮挡、脸太小、运动拖影严重的帧。
4. 将保留帧按用途移动到身份脸、上半身、全身、服装、发型、妆容、姿势或场景目录。
5. 连续相似帧只保留 1 到 2 张。
6. 不确定但暂时保留的素材放入 `rejected`，不要混入训练候选集。

## 上半身、下半身、全身图分别如何使用

### 上半身图

上半身图通常最适合身份 LoRA，因为它同时包含脸、发型、眼镜、肩颈、上衣和气质。高质量上半身图可放入 `selected_upper_body`，并作为身份 LoRA 候选。

### 下半身图

下半身图主要用于裤裙、鞋子、包、腿部姿势和配饰参考。因为脸部通常不可见或太小，不建议用于身份 LoRA。

### 全身图

全身图适合控制身体比例、整体穿搭、姿势和构图。如果脸部足够清晰，可以少量作为身份 LoRA 候选；如果脸部太小，应放入 `selected_full_body`、`outfit_refs` 或 `pose_refs`。

## 哪些素材不能用于身份 LoRA

不要用于身份 LoRA 的素材：

- 脸部模糊、脸太小或五官不可辨认。
- 大侧脸、背影、低头、遮脸、闭眼。
- 眼镜严重变形或反光完全遮住眼睛。
- 强滤镜、美颜过度、肤色失真。
- 多人合照且主体不明确。
- 特殊妆容、特殊发型或临时造型与常设角色冲突。
- 重点是服装、鞋包、姿势或场景，但身份信息不足。
- 带水印、字幕、贴纸、UI 遮挡的图片。
- 与目标年龄感、脸型、发色或气质明显冲突的图片。
